# Flexbox Photo Gallery
Responsive photo gallery built with Flexbox, entailing a card layout with hover effects.

---

## Live Site
https://diegomartins8.github.io/flexbox-photo-gallery/ 

---

## Tools Used
- HTML5
- CSS3 (Flexbox)

---

## Description
This website was created for a Web Design course assignment, with intention to demonstrate responsive layout design using CSS Flexbox. 
The gallery showcases 8 image cards that automatically adjust to screen sizes, and includes hover effects for enhanced UX.

---

## Skills
- Flexbox layout and responsive design
- Image based card UI components
- CSS hover and transition effects
- Front-end layout optimization
